# nb2pdf_agent 📓→📄

An AI-powered agent that converts Jupyter Notebook (`.ipynb`) files into professionally
formatted, ready-to-submit PDF lab reports — automatically.

---

## Features

| Capability | Details |
|---|---|
| **Markdown rendering** | Headings H1–H4, bold, italic, lists, tables, blockquotes, inline code |
| **Code cells** | Styled Python blocks with `In [N]:` labels and accent left-border |
| **Cell outputs** | Styled output blocks with `Out [N]:` labels, stream text, images |
| **Error tracebacks** | Red-highlighted error blocks with full traceback text |
| **Cover page** | Dark-themed cover with title, kernel info, and cell count stats |
| **Table of contents** | Auto-generated from heading structure with anchor links |
| **Header / footer** | Document title + date in header, page N of M in footer |

---

## Setup

### Step 1 — Install Python dependencies
```bash
pip install nbformat reportlab pygments Pillow
```

### Step 2 — Place the script

Save `nb2pdf_agent.py` in your working directory. No installation needed.

---

## Usage

### Basic usage
```bash
python nb2pdf_agent.py my_notebook.ipynb
# Output saved as: my_notebook_report.pdf
```

### Custom output path
```bash
python nb2pdf_agent.py my_notebook.ipynb -o reports/lab3_report.pdf
```

### Help
```bash
python nb2pdf_agent.py --help
```

---

## Output Structure

Every generated PDF includes:

1. **Cover Page** — Title, kernel version, date, total/code/markdown cell counts
2. **Table of Contents** — Auto-extracted from all `#` headings
3. **Content Pages** — All cells in notebook order (markdown → code → output)
4. **Header** — Document title and date on every page
5. **Footer** — "Generated by nb2pdf_agent" + page `N of M`

## How It Works
```
your_notebook.ipynb
        │
        ▼
  nbformat.read()          Parse notebook JSON structure
        │
        ├── Markdown cells ──► parse_markdown()
        │                      Headings, lists, tables, blockquotes
        │
        ├── Code cells ──────► render_code_block()
        │                      Styled box with In[N]: label
        │
        └── Cell outputs ────► render_outputs()
                               Text, images, error tracebacks
        │
        ▼
  ReportLab PDF engine
        │
        ├── Cover page
        ├── Table of Contents
        ├── Content pages
        └── Header + Footer on every page
        │
        ▼
  your_notebook_report.pdf  ✅
```

## Dependencies

| Package | Version | Purpose |
|---|---|---|
| `nbformat` | ≥ 5.0 | Parse `.ipynb` JSON |
| `reportlab` | ≥ 4.0 | Generate PDF layouts |
| `pygments` | ≥ 2.0 | Code syntax support |
| `Pillow` | ≥ 9.0 | Decode base64 image outputs |

## Limitations

- LaTeX math (`$...$`) renders as plain text (no equation rendering)
- ipywidgets outputs are skipped
- Outputs longer than 80 lines are truncated with a notice
